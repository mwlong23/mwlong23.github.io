Code Matters.


Welcome! I'm glad you're here. My name is Mitch & I'm a coder. I enjoy building web applications, games and computer graphics. I recently graduated from Epicodus, a vocational school for computer programmers.

There I built a solid foundation in Javascript, Angular 5, Ruby & Rails. I also spent less time debugging by utilizing Test Driven Development(TDD).

Less debugging means more fun adding new features.

Shortly after falling in love with programming, I found Javascript. Since then I've been working feverishly to deepen my knowledge of the language and frameworks. I'm enrolled in three courses where I'm diving deep into es6/es7, Angular 5, NodeJS and Express. What an exciting time to be a developer!
